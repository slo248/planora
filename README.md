# Planora Monorepo

Planora is a task management system. This monorepo contains both the backend API and the frontend web application for managing tasks efficiently.

This repository contains two main projects:


## planora-api
Backend API for task management, built with NestJS. Located in the `planora-api/` folder.

### Usage
```sh
cd planora-api
pnpm install
pnpm start
```


## planora-web
Frontend web application for managing tasks, built with React and Vite. Located in the `planora-web/` folder.

### Usage
```sh
cd planora-web
pnpm install
pnpm run dev
```
